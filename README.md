# GitTracker

Track your repository status!


## Setup

Go to *repos.db* and write:

```	
first line = number of repositories
repository name
repository path
```

**Exemple**: *template.db*.

## Compile & run

Open a terminal on the source folder and compile the code with:

```
g++ -o Main Main.cpp
```

To run the program just:

```C++
Main // Windows
./Main // Linux
```
