# GitTracker

Track your repository status!

## Setup

Open *repos.db* and write:

```	
repository 1 path
repository 2 path
```

**Exemple at *template.db***.

## Compile & run

Open a terminal on the source folder and compile the code with:

```
g++ -o Main Main.cpp
```

To run the program run: "Main.exe" (windows) or "./Main" (linux)
